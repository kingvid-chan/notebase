# 0.0.1 — 笔记管理系统初始交付

## 范围

老板需求（e000001）：开发一个笔记管理系统，支持用户登录注册、笔记管理（增删改查、搜索、标签分类、Markdown渲染）、笔记公开分享链接。

完整需求见 e000002（requirement_defined），老板确认见 e000003（owner_confirmation）。

## 业务模型

- User: 注册用户（id, username, email, password_hash, created_at, updated_at）
- Note: 笔记主体（id, user_id, title, content_markdown, content_html, is_public, created_at, updated_at）
- Tag: 用户自定义标签（id, name, user_id, created_at）
- NoteTag: 笔记标签多对多关联（note_id, tag_id）
- ShareLink: 公开分享链接（id, note_id, token, expires_at, created_at）

关系：User 1:N Note; Note N:M Tag via NoteTag; Note 1:N ShareLink

## 产品流程

- 注册/登录/登出（演示账号 + 假数据）
- 笔记 CRUD + Markdown 实时预览
- 标签管理（添加/移除/筛选）
- 全文搜索
- 公开分享链接生成与只读访问
- 错误路径：未登录拦截、权限隔离、无效分享链接提示

## 明确不做（本版）

- OAuth/第三方登录
- 协作编辑
- 文件附件上传
- 邮件验证/密码重置
- 移动端原生 App
